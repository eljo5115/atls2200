Kraken Automated Trading Interface

The Kraken Automated Trading Interface started as an experiment with python to test what was really possible with algorithmic trading. The decision making was super simple as I don't know how to implement an AI model of the data. The main focus of the early stage was to understand how the API was used. The next step will be to deploy a web app, which I am working on here.

Genius Lyrics API

As I started into web development the idea of scraping websites for specific data was presented to me. I decided an useful way of using this was to improve upon Genius's pre-existing API by making an API that scrapes Genius lyrics, and just the lyrics. Further implementation will be an interactive word web that generates from the lyrics. Written in Python with fastapi.

eZ Trade API

eZ Trade API goes hand in hand with the Kraken Automated Trading Interface and serves as the backbone in deciding trades and positions. eZ Trade has a public endpoint that you can access by creating an account here.
