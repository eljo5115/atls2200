Bitcoin Trading Bot

I've been working on a python script that will trade Bitcoin for me. Due to the high volatility of the market, I think this is a viable strategy for making passive income. I've published the code onto Github and am currently running a trial with only $100. It wouldn't take much to change the code to buy with much more money and the results would be scalable. This trading strategy basically relies on the volatility of the asset and results could vary from day to day. The end result would actually be a program that shows account balance, open orders, price charts, trades various assets and probably some sort of GUI. Currently I just have a command line script with a while loop calling someone else's analysis API and trading through the Kraken API. What I have is a rudimentary proof of concept, but using the power of AI this could be much better at predicting the future price and buying and selling.

I'll keep updating the code in this Github repo as I add features and iron out any bugs.

A Silly Solution to a Simple Problem

Wider aisles might fix the problem, or having an app with store inventory, but that doesn't fit the Trader Joe's brand. Trader Joe's needs a solution that almost doesn't make sense to people except their most loyal customers, like the constant need to change their product selections.
